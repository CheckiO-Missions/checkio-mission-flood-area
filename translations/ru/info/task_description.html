В этой задаче требуется оценить ущерб от наводнения.

Даётся схема поперечного сечения участка ландшафта (в виде строки). Символы '/' и '\' представляют собой наклонные участки ландашфта, '_' - плоские. Приведённая на изображении ниже область описывается строкой "/\\///\_/\/\\\\/_/\\///__\\\_\\/_\/_/".

Считается, что дождь идёт непрерывно и затапливает все низменности, а с обеих сторон каждой из схем находится море, куда стекает лишняя вода. В примере ниже дождь затопит все низины, и количество затопленных клеток составит 4, 2, 1, 19 и 9.

Написанная функция должна возвращать список (или итератор) с количеством затопленных клеток для каждой из низин, слева направо. Если затопленных клеток нет, необходимо вернуть [] (или пустой итератор).
